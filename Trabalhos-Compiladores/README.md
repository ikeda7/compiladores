# Compiladores

Para que o projeto funcione é necessario ter o python ou python3 instalado e a biblioteca TKinter

Para executar o projeto:
1. `python -m venv venv`
2. Bash: `source venv/bin/activate` | `venv/Scripts/activate.bat`
3. Instalar o TKinter
4. `pip install -r requirements.txt`
5. `(python | python3) src/Main.py`

# Estrutura

O projeto contem as seguintes fases e estruturas:

1. Primeira entrega: arquivos do analisador léxico (calculadora)
2. Segunda entrega: arquivos do analisador léxico para LALG
3. Terceira entrega: arquivos do analisador sintático
4. Quarta entrega: Análise Semântica + Geração/interpretação de código
